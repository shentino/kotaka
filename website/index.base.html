Administrative contacts

For general chat, or to report anything that cannot be reported any other way, use shentino@gmail.com

To report mud problems, use the bug, idea, typo, and nohelp commands. If they don't work, or you cannot log in, use shentino@gmail.com

To report message board issues (abuse/problems), PM shentino or admin on the forums. If that doesn't work, contact shentino@gmail.com
